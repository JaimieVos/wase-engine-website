Wase Engine

Wase Engine is an open source 2d game library made with C++ and the SDL libraries. Wase Engine can be used to create all different kinds of 2d games. The main goal for Wase Engine is to give you the tools you need for creating games while keeping as much control as possible.

Documentation

To found out more about how Wase Engine works you can head over to the documentation page.

Contributing

Wase Engine is completely open source and free to use. We love it when other people help and contribute to the source code of Wase Engine. To start contributing to Wase Engine you should read the contributing page.
